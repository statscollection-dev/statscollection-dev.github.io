Online¶

Online algorithms are algorithms which process a data stream of unknown length. These algorithms become relevant when data is too large to fit into memory.

Overview¶

The following classes are available for online computations on data streams, i.e. any iterable object in Python.

Mean

The arithmetic mean.

Min

The minimum.

Max

The maximum.

Online algorithms for sampling.

Sample

Sample with equal probability from an iterable of unknown length.

Tutorial¶

The classes in this module share a common api, illustrated in the following example.

>>> from statscollection.online import Mean
>>> mean = Mean().fit(2.0)  # Fit a single item
>>> print(mean.evaluate())
2.0
>>> mean = mean.fit(iter([5.0, 2.0]))  # Fit an iterable object
>>> print(mean.evaluate())
3.0

It’s possible to…

>>> for result in Mean().yield_from(iter([1.0, 3.0, 5.0])):
...     print(result)
1.0
2.0
3.0

>>> print('hello world')
hello world
